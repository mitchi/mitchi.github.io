            Extended Combinatorial Testing using Graph Algorithms and Apache Spark

This thesis presents a generalization of t-way testing, reductions to graph coloring and vertex cover problems and distributed algorithms using Apache Spark. Experimental results were obtained using computer clusters provided by ComputeCanada.
